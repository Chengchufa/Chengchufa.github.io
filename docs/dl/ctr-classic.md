## 章节大纲

- CTR 问题定义：显示 vs 隐式反馈、采样偏差、冷启动
- LR（逻辑回归）：工业界最早期大规模使用
- POLY2：二阶特征组合，参数量爆炸
- FM（因子分解机）：每个特征学隐向量，稀疏鲁棒
- FFM：域感知 FM，每个域独立隐向量
- GBDT+LR（Facebook 方案）：树作离散化 + 特征交叉
- FTRL：谷歌广告用的在线优化器
- 特征工程核心（工业级 CTR 传统派）：
  - 用户画像特征（年龄、性别、历史行为）
  - 物品画像、内容特征
  - 上下文特征（时间、设备、场景）
  - 交叉特征 + 统计特征（滑窗 CTR、CVR）
- 评估指标：AUC、RIG、LogLoss、校准度
- Kaggle Criteo / Avazu 竞赛方案复盘
