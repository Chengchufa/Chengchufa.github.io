## 💡 推荐工具清单

### AI 辅助编程
- **Claude Code** - 终端集成，大上下文，复杂重构首选
- **Cursor / Windsurf** - IDE 类 AI 编程工具，适合日常编辑
- **GitHub Copilot** - 补全速度快，与 VS Code 深度集成

### 笔记管理
- **Obsidian** - 本地优先、双链、可扩展插件生态
- **Typora / Zotero** - 阅读与文献管理
- **Mermaid / Draw.io** - 绘图与流程图

### 开发效率
- **fzf / ripgrep / fd** - 模糊搜索、文本搜索、文件查找
- **just / make** - 项目命令脚本管理
- **direnv / nvm / fnm** - 环境版本切换
