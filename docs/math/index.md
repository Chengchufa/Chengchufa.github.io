# 数学基础

机器学习和深度学习的核心是数学。本模块**计划系统梳理**从事算法工作必备的数学基础知识，
包括线性代数、概率论、数值计算以及蒙特卡洛采样方法等。每章完成后会包含
**核心定义、关键定理推导、典型例题、与机器学习的关联应用**四个板块，建议配合具体算法章节同步学习。

> 🎯 **当前进度**：4 章规划中，已完成 1 章，其余「待补」——开始学习后会按顺序逐篇追加。

## 🎯 学习建议

1. **零基础入门**：建议按 1→2→3→4 的顺序学习，第 1、2 章是核心中的核心。
2. **已有基础**：可直接跳到对应章节，也可仅阅读「与机器学习的关联」小节。
3. **配合实践**：每学完一章，建议用 NumPy 手动实现关键公式加深理解。
